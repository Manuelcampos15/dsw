Propósito del sitio

El objetivo es aprender qué es una regla de estilo CSS, cuál es su sintaxis CSS correcta, además aprenderemos a utilizar los selectores, muy importantes para apuntar o seleccionar elementos HTML específicos y estilizarlos con CSS.

HTML es un lenguaje utilizado para describir páginas web

HTML se refiere al lenguaje de marcado de hipertexto

HTML no es un lenguaje de programación, sino un lenguaje de marcado

HTML utiliza etiquetas de marcado para describir páginas web

HTML generalmente aparece en pares, y la mayoría de las etiquetas también aparecen en pares
